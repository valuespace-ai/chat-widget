ValueSpace Chat Widget - Simple Usage

This page demonstrates the most basic integration of the ValueSpace Chat Widget. The widget is loaded with minimal configuration and without any test buttons or advanced features.

How to Integrate

To add the chat widget to your website, simply include the script and initialize it with your configuration:

<!-- 1. Include the chat widget script -->
<script src="path/to/vs-chat-widget.iife.js"></script>

<!-- 2. Initialize the widget with your configuration -->
<script>
  window.initChatWidget({
    botServiceUrl: 'http://localhost:3978',
    userName: 'User',
    tenantId: 'default'
  });
</script>

Configuration Options

The most important configuration options are:

  • botServiceUrl: URL to your bot service
  • userName: Name of the user (optional)
  • tenantId: Your tenant identifier (optional)
  • customStyles: Object containing style customizations (optional)

For more advanced configuration options, see the documentation or the advanced example.
